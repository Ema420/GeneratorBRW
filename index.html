      Come funziona

      1. Seleziona il numero di giocatori e le espansioni che possiedi o vuoi usare.
      2. Clicca su “Genera Setup” per ottenere:

      • Un mago casuale per ogni giocatore, pescato da tutte le espansioni selezionate.
      • Un set di scuole di magia per ciascun mago (es. 2 per giocatore).
      • Un set di stanze casuali per la Loggia, composte dalle espansioni scelte.
      • Una modalità / scenario suggerito.

      Puoi modificare liberamente i contenuti delle espansioni nel file script.js (maghi, scuole, stanze) per adattare il tool alla tua collezione.

      Fan-made tool per uso personale, non ufficiale né affiliato con Ludus Magnus Studio.
