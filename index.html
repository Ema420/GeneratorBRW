      Come funziona

      Questo generatore crea casualmente:

      • Un mago diverso per ogni giocatore
      • Un set di scuole di magia per ciascun mago
      • Un set di stanze casuali per la Loggia
      • Una modalità / scenario suggerito

      Puoi modificare il contenuto delle liste (maghi, scuole, stanze, scenari) nel file script.js per adattarlo alle tue espansioni.

      Fan-made tool per uso personale, non ufficiale né affiliato con Ludus Magnus Studio.
